# KozFlow

Static landing page. AI-агенты, автоматизация, дизайн.

## Stack

Pure HTML / CSS / JS. GSAP for nav menu. No build step.

## Local dev

```sh
python -m http.server 8000
# → http://localhost:8000/
```

## Deploy

Deploys via Vercel on push to `main`.
